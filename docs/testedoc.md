# Guia do cliente

Aqui encontramos as informações do cliente importantes sobre o hotel

## Como fazer uma reserva

siga os passos abaixo:

1. Escolha as datas da estadia
2. Selecione o tipo de quarto
3. Preencha os seus dados
4. Confirme o pagamento
5. Aguarde o email de comfirmação

### Registar Check-in

A aplicação permite registar o check-in de um cliente.

Passos:

1. Aceda ao menu **check-in**
2. Selecione a reserva do cliente
3. Verifique os dados apresentados
4. Clique no botão **Confirmar Check-in**

Após a confirmação, o estado da reserva muda para **Ativa**.

## Check-out

**Horario:** até às **12h**

1.Arume os seus pertences
2.Entrege a chave na recessão
3.Confirme se não há pendencias

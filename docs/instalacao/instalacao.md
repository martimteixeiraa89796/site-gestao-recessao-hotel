# Instalação da aplicação

A nossa aplicação é de fácil instalação e portátil, pelo que apenas necessita de um descompactador de ficheiros para poder utilizá-la.

## Descarregar Aplicação

A aplicação pode ser descarregada a partir da [página de downloads](/download) do nosso site e está disponível para grande parte de sistemas operativos.

A aplicação é descarregada num ficheiro `.zip`.

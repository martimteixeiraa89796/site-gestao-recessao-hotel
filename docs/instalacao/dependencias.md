# Dependências

A nossa aplicação foi desenhada para ser leve no sistema, mas mesmo assim a mesma depende de certos fatores.

## Resumo Geral

A nossa aplicação depende de:
- Terminal Emulator --> Para imprimir para o ecrã;
- Python --> Linguagem em que a aplicação foi escrita;
- 7zip ou outro --> Descompactar aplicação;

## Explicação

A nossa aplicação pode ser utilizada na maioria das plataformas atuais que suportam a instalação da linguagem de programação **Python**. Antes de poder utilizar a nossa aplicação, deverá proceder à sua instalação. Isto é o resultado do facto de que Python não pode ser compilado como outras linguagens, mas sim interpretado linha por linha em tempo real.

A aplicação necessita de uma interface que possa correr aplicações CLI. A nossa aplicação foi desenvolvida para correr num ambiente sem capacidades gráficas.

A aplicação é descarregada num ficheiro `.zip`, pelo que é necessário descompactá-la.

# Inseir Funcionários

Após a definição das funções que são precisas para cada funcionário, é preciso então tratar de arranjar os funcionários em si.

## Tabela Funcionários

A tabela funçionário é onde será colucado os dados dos funcionários que trabalham na recessão do hotel 

Esta tabela é dependente da tabela funções e necessária para as tabelas Horário e Reserva

### adicionar dados

Para adicionar dados à tabela basta abrir o menu da aplicação e selecionar a função de adicionar dados.

![Menu inicial da aplicação](/img/aplicacao/menu_inicial.png)

Após selecionar a opção de adicionar dados basta selecionar a tabela Funcionários

7. Tb_Funcionarios

![lista de escolha](/img/aplicacao/inserir_dados.png)

Por fim adicione os dados de acordo com o que lhe é pedido

![Inserção de funcionários](/img/aplicacao/insercao_dados/funcionario.png)

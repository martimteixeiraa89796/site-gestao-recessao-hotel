# Inseir Reservas

## Tabela Reserva

A tabela reserva é onde será colucado os dados das reservas futuras do hotel

Esta tabela é dependente das tabelas funçionarios e tipo_reserva e necessária para a tabela Hospedes

### adiciunar dados

Para adicionar dados à tabela basta abrir o menu da aplicação e selecionar a função de adicionar dados.

imagem

Após selecionar a opção de adicionar dados basta selecionar a tabela Hospedes

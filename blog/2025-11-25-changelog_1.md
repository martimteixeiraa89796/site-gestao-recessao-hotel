Uma nova versão estável foi lançada!

Esta nova versão traz as funcionalidade básicas, como visualizar, alterar ou apagar dados.

<!-- truncate -->

Esta nova versão foi desenvolvida com um pensamento modular, ou seja, cada parte da nossa aplicação
foi desenhada para funcionar de forma isolada, de modo a preparar para futuras adições e manutenção do código.

Módulos desenvolvidos:

- ferramentas_BD
- funcoes_sqlite_create
- funcoes_sqlite_delte
- funcoes_sqlite_insert
- funcoes_sqlite_select
- funcoes_sqlite_update
- main

Cada módulo da aplicação pode ser corrido manualmente se as respetivas funções forem chamadas, mas a equipa criou um UI básico
para essa mesma função.
